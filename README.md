# RocketTele
Arduino based live rocket telemetry and recording.

Goals:
Logging frequency 10Hz to SD card:
	Acceleration data.
	Baro pressure.
	GPS.
TX via serial to ground RX 
	Altitude.
	GPS.

I2Cdev and MPU6050 code attributed to jrowberg.
